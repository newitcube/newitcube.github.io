# newitcube.github.io


Чтобы запустить сайт `tokazin.github.io`, загрузите файлы (обязательно `index.html`) в репозиторий через браузер или `git push`. GitHub автоматически опубликует сайт по этому адресу. Убедитесь в настройках (`Settings` -> `Pages`), что источником выбрана ветка `main` или `master`. 

**Пошаговая инструкция:**

1. **Подготовьте файлы:** Убедитесь, что главная страница называется **`index.html`** и находится в корне репозитория.
2. **Загрузите файлы:**
   - **Через сайт (простой способ):** На странице репозитория нажмите "Add file" -> "Upload files" и перетащите ваши HTML/CSS/JS файлы.
   - **Через Git:** Используйте команды `git add .`, `git commit -m "site"`, `git push`.
3. **Активация (обычно автоматически):**
   - Перейдите в **Settings** (Настройки) вашего репозитория.
   - В левом меню найдите раздел **Pages**.
   - Убедитесь, что в поле "Source" выбрана ветка **main** (или master) и папка **/(root)**, затем нажмите **Save**.
4. **Проверка:** Через 1-5 минут сайт будет доступен по адресу `https://tokazin.github.io`. 

*Примечание: GitHub Pages поддерживает только статические сайты (HTML, CSS, JS).*
